Cursos gratis

HTML es el lenguaje de marcado para páginas web. Con él puedes crear y estructurar sitios web y aplicaciones web. HTML describe la estructura de una página web semánticamente y originalmente incluía cues para la visualización de la página en navegadores web.

CSS es un lenguaje de hojas de estilo que describe el diseño visual de un documento escrito en un lenguaje de marcado. CSS describe cómo debe ser mostrado un documento HTML o XML (incluyendo XML dialectos como SVG o XHTML). CSS no es un lenguaje de programación, y no tiene ningún mecanismo para interactuar con los datos o con el control de flujo, por lo que no es posible crear aplicaciones con CSS.

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos,​ basado en prototipos, imperativo, débilmente tipado y dinámico.
